An end-to-end data analytics project analyzing over **1,000,000+ Amazon e-commerce transactions**. This project covers automated data cleaning and transformation in Python (Jupyter Notebooks) and interactive business intelligence reporting in Power BI.

---

## 📌 Business Case & Objectives
The goal of this analysis is to evaluate sales performance, customer purchasing behavior, fulfillment efficiency, and product profitability across global e-commerce operations. 

**Key Questions Addressed:**
* What are the primary revenue drivers across product categories?
* How do fulfillment methods (FBA vs. Merchant) impact order delivery performance and return rates?
* What seasonal trends exist in customer purchasing patterns over time?
